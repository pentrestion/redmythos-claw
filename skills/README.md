# REDMYTHOS CLAW 🦀 Skills Directory

Drop `.md` or `.txt` files here to add specialized knowledge or procedures to the agent.
The agent will automatically read these files as part of its system prompt if configured in the mode.

## Examples
- `security_audit.md`: Best practices for auditing code.
- `saas_blueprint.md`: Architecture for multi-tenant apps.
- `termux_hacks.txt`: Collection of useful Termux commands.
